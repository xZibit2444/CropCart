Setup Instructions

  1. Go to app.supabase.com and log in
  2. Navigate to your project (CropCart)
  3. Click on SQL Editor in the left sidebar
  4. Click New Query
  5. Copy the SQL code below
  6. Paste it into the SQL editor
  7. Click Run to execute the query
